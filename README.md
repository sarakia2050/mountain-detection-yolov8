# mountain-detection-yolov8
A practical Computer Vision project for mountain detection using YOLOv8, CVAT, Roboflow, and Ultralytics.

### Manual Annotation Example

This image shows one of my first manually labeled mountain examples in CVAT.

![Manual Annotation](images/manual_annotation_cvat.jpg)

### Prediction on a New Image

After training with a larger dataset, the model was tested on a different mountain image and detected it with 88% confidence.
